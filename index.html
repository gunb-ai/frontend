daglang · gunbc compiler

One graph. Many targets.

daglang represents software systems as typed graphs. gunbc analyzes the graph, runs structural checks, and emits artifacts from the same source. Start with the compiler today; follow the roadmap toward full-stack omni-emission, agent-native editing, and non-code targets.

Lead example · available now

One declaration of Condition, every consumer in lockstep.

Real source: dsl/examples/weather/weather.dag. Verified today on the Rust target.

weather.dag — shared domain Available now · Rust 
type Forecast {
  location: String
  high: Temperature
  low: Temperature
  condition: Condition
}

type Condition
  = Sunny | Cloudy
  | Rainy { mm_per_hour: Float }
  | Snowy { cm_per_hour: Float }
→ Rust types emitted and checked with cargo check
→ Add a variant: every match that doesn't cover it fails closed
→ Go and Python stabilizing on the same graph

cd daglang && cargo build --release -p v2-compiler --bin gunbc
./target/release/gunbc --help
