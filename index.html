gunb.ai gunbc · structural compiler

A structural compiler for composable systems.

Describe the system once. Let the compiler derive the rest.

Modern systems are full of hand-written translations: backend types into client types, models into schemas, services into docs, code into tests, source into build rules.

daglang gives the system one structural description. gunbc checks that description and derives the mechanical pieces.

When the description is incomplete, the compiler refuses to guess.
