# gunb.ai frontend

Prototype assets for the Gunbai landing page.

- **`moodboard.html`** — Single-file theme board: palette tokens, typography specimens, ASCII motifs (branch + pipeline), interactive node strip, hero/error prototypes. **Layout notes in-file:** *left → right = time* where applicable; *grow down* (main first, branches / side material below on scroll). Open directly in a browser (`file://` is fine).

Reference compiler repo: `../gunbc`.
