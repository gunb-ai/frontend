primitives · 01

Materials

Five materials. Each has a reason. Moss is structural continuity, not "success." Clay is a boundary, not "danger." Neither is a UI status color — no traffic-light usage anywhere in the system.

Voidmaterial

The ground. Near-black, slightly warm. Product surface.

Stonematerial

Warm grey. Primary text on dark. Patient, recessive.

Warm Whitematerial

Off-white, warm cast. Light-mode surface.

Mossaccent

Structural continuity. Link, held relation. Not "success."

Clayaccent

Boundary. Diagnostic, invariant break. Not "danger."

primitives · 13

Out of scope

Primitives we deliberately don't have. Each is a small commitment to the moodboard's register. If a future surface needs one of these, it justifies the addition against the rule it would violate.

✕Filled CTA buttonUse text action with arrow. Fills read as SaaS.
✕Status badges"Available now / Coming soon" → docs, not UI.
✕Success / warning / info calloutsSystem has no traffic-light colors.
✕TabsIf content needs tabs, it's two pages.
✕Modal / dialogPages don't interrupt themselves.
✕Toast / notificationCompiler doesn't shout.
✕TooltipBrowser default title only.
✕Accordion / expanderIf it's collapsible, it's optional. If it's optional, why show it.
✕Carousel / sliderTime-gated content hides itself.
✕Avatar / user chipNo identity surface in the public site.
✕Progress barReceipts show completion, not animation.
✕Gradient / glowMaterials, not effects.
✕Iconography packOne mark. No icon set.
✕Drop shadowBorders, not depth.
✕Hero video / animationNo motion until V2 (rule 6).
✕Feature card gridThe drift vector. See PROPOSAL.md.

gunbc · structural compiler

A structural compiler for composable systems.

Describe the system once. Let the compiler derive the rest.

Modern systems are full of hand-written translations: backend types to client types, models to schemas, services to docs, code to tests, source to build rules.

daglang describes the system once. gunbc checks that description and derives the mechanical pieces. When the description is incomplete, the compiler refuses to guess.
